Font Awesome Icons

Shortcode for Font Awesome is very simillar to the buttons shortcode. It looks like that: [fa]
There is 3 options with different attributes - icon, href and target.

Icon: You can use any FontAwesome icon. And there is 479 of them. You must use just title of the icon. Example: [fa icon="fa-phone"]
Href: You can add URL to the icon. Example: [fa href="http://www.proteusthemes.com"]
Target: You can choose if you want to open link in the same (_self) or new - (_blank) window. Example: [fa target="_blank"]
